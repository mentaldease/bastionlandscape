History

2.0.1 [06/02/2005]
Bug fixes to the ODBC driver.
2.0.0 [03/22/2005]
Added ADO driver and some bug fixes to the ODBC driver.
2.0 beta 3 [12/23/2004]
Minor bug fixes.
2.0 beta 2 [11/26/2004]
Bug fixes and the new SQLite and JDBC drivers. Also, it follows the package model for Lua 5.1 (see section Installation for more details).
2.0 beta [12/10/2003]

Version 2.0 has some design modifications and implementation improvements

  • New fetch method: more eficient and more flexible
  • New setautocommit method
  • Lua 5.0 and 5.1 compatible
  • Dynamically loadable or statically linked
  • New drivers for Oracle and MySQL databases
